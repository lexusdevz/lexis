## 🔧 Troubleshooting & Common Issues

Even with a smooth setup, issues can arise. Here are the most frequently reported problems and their solutions:

**❓ The menu doesn't open after pressing F4**
→ Confirm that Secure Boot is disabled in BIOS. Restart your PC after disabling it. Re-launch the injector as Administrator, then launch the game.

**❓ The injector shows "Not Injected" status**
→ Make sure the game is fully loaded (past the main menu). Try switching from Enhanced to Legacy mode in the launcher settings. Verify your license key is active on the portal.

**❓ My antivirus deleted the loader file**
→ Add the LexisMenu folder to your antivirus exclusion list. Re-download the package from the official site after adding the exception.

**❓ The menu freezes or causes the game to crash**
→ Disable overlays from other applications (Discord overlay, GeForce Experience, Steam overlay). Launch in `--debug` mode and check the log file for error codes.

**❓ ESP isn't showing other players**
→ Verify `esp_enabled` is set to `true` in config. Some session types may restrict visibility — try a different lobby.

**❓ Money loop stops working mid-session**
→ This is often caused by the game's own server-side validation. Lower the loop amount per cycle (try $250,000 instead of $1,000,000) and add a 5-second delay between iterations in settings.

**❓ Config resets after every restart**
→ Check that the config folder has proper write permissions. Right-click the AppData/Roaming/LexisMenu folder → Properties → Security → Grant full control to your user account.

**❓ Features not working after game update**
→ Wait for the official patch from the Lexis team (usually deployed within 24–72 hours of a game update). Check the Discord/official channels for status updates.
